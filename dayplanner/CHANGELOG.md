# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-11-10

### 新增功能

#### 日历视图
- 月历形式显示当前月份的所有日期
- 有任务的日期显示视觉标识（蓝色标记）
- 点击日期切换到对应的任务编辑视图
- 上一月/下一月导航按钮

#### 任务编辑
- Markdown 文本编辑器
- 支持常用 Markdown 语法（标题、列表、粗体、斜体、代码块等）
- 实时 Markdown 预览
- 自动保存功能（停止输入 2 秒后保存）
- 任务数据本地存储（JSON 格式）

#### 企业微信提醒
- 可配置的每日提醒时间
- 通过企业微信 Webhook 发送提醒消息
- 检测当天是否有任务记录
- 防重复发送机制（每天只发送一次）
- 可开启/关闭提醒功能

#### 设置界面
- 企业微信 Webhook 地址配置
- 提醒时间设置（24小时格式）
- 提醒开关
- 配置验证和保存

#### 系统功能
- 分层架构设计（UI层、业务逻辑层、数据访问层）
- 完整的错误处理和日志记录
- 自动创建必要的目录和配置文件
- Windows 原生桌面应用（无控制台窗口）

### 技术实现

- **GUI 框架**: Fyne v2
- **Markdown 渲染**: goldmark
- **数据存储**: JSON 文件
- **编程语言**: Go 1.21+
- **目标平台**: Windows 10+

### 文件结构

```
daily-report-tool/
├── cmd/daily-report/          # 应用程序入口
├── internal/
│   ├── ui/                    # UI 组件
│   ├── service/               # 业务逻辑
│   ├── repository/            # 数据访问
│   ├── model/                 # 数据模型
│   └── util/                  # 工具函数
├── config/                    # 配置文件
├── data/tasks/                # 任务数据
└── logs/                      # 日志文件
```
