# Windows 右键打开命令行工具 - 使用说明

## 📦 文件说明

- `opencmd.exe` - Windows 可执行文件（1.9MB）

## 🚀 快速开始

### 第一步：复制文件到合适位置

建议将 `opencmd.exe` 复制到一个固定的位置，例如：
```
C:\Program Files\OpenCmd\opencmd.exe
```

⚠️ **重要**：不要随意移动或删除此文件，否则右键菜单将无法工作。

### 第二步：安装右键菜单

1. 按 `Win + X`，选择"终端管理员"或"命令提示符（管理员）"
2. 切换到 opencmd.exe 所在目录，例如：
   ```cmd
   cd "C:\Program Files\OpenCmd"
   ```
3. 执行安装命令：
   ```cmd
   opencmd.exe install
   ```
4. 看到"右键菜单安装成功！"提示即可

### 第三步：使用右键菜单

安装成功后，你可以：

1. **在文件夹内右键**：打开任意文件夹，在空白处右键，选择"在此处打开命令行"
2. **在文件夹上右键**：在文件夹图标上右键，选择"在此处打开命令行"

命令提示符会自动在该目录打开。

## 🗑️ 卸载

如果不再需要此功能：

1. 以管理员身份打开命令提示符
2. 切换到 opencmd.exe 所在目录
3. 执行：
   ```cmd
   opencmd.exe uninstall
   ```
4. 看到"右键菜单已卸载"提示即可
5. 然后可以安全删除 opencmd.exe 文件

## ❓ 常见问题

### Q: 为什么需要管理员权限？
A: 因为需要修改 Windows 注册表来添加右键菜单项。

### Q: 安装后没有看到右键菜单？
A: 请确认：
- 是否以管理员身份运行安装命令
- 是否看到"安装成功"的提示
- 尝试重启资源管理器（任务管理器 → 找到"Windows 资源管理器" → 重新启动）

### Q: 移动了 opencmd.exe 文件后右键菜单不工作？
A: 需要先卸载，然后在新位置重新安装。

### Q: 支持 PowerShell 吗？
A: 当前版本打开的是 cmd.exe（命令提示符）。如需 PowerShell 版本，可以修改源代码。

### Q: 安全吗？
A: 完全安全。源代码开放，仅修改注册表添加右键菜单项，不涉及任何恶意操作。

## 📝 技术细节

- 修改的注册表位置：
  - `HKEY_CLASSES_ROOT\Directory\Background\shell\OpenCmdHere`
  - `HKEY_CLASSES_ROOT\Directory\shell\OpenCmdHere`
- 使用 cmd.exe 的图标
- 命令格式：`cmd.exe /K "cd /d <目录路径>"`

- 卸载前请确保以管理员身份运行
